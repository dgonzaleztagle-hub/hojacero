# BRAND_SOUL - Biocrom 2 (Web-Only)

## Philosophy
Precision operativa para laboratorios que no pueden detenerse.

## Genetic Code (100%)
- 45% Institutional Trust: autoridad tecnica, claridad y sobriedad.
- 35% Analytical Engineering: metodo, continuidad, control y trazabilidad.
- 20% Service Velocity: contacto directo, respuesta util y accionable.

## Visual Signals

| `#D10A2A` | Signal Red | Accion / foco critico | 23% |

### Typography Strategy
- Headings (H1-H2): `Space Grotesk` 700-800 | Autoridad moderna y tecnica.
- Body/Tech (P/Small): `Inter` 400-500 | Lectura limpia en desktop/mobile.

### Key Visual Motifs (Must Include)
- [x] Grid tecnico sutil (lineas / nodos de red).
- [x] Tarjetas con jerarquia asimetrica y acentos rojos.
- [x] Micro-motion funcional (sin ruido estetico).

1. Hero con promesa de continuidad analitica.
